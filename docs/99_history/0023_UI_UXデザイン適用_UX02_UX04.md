# 0023 UI/UX デザイン適用（UX-02, UX-04）

## 日付

2025-02-07

## 概要

UX-04 デザイントークンと UX-02 CP-000/CP-001 に沿い、ダークテーマ・ロゴ・ファビコン・ホーム CTA・一覧カード・ボタンを統一した。

## 変更ファイル一覧

- src/app/globals.css: `:root` にカラー・スペーシング・角丸・シャドウ・tile/center-block トークン定義。`body` に背景・文字色・フォント変数適用。
- tailwind.config.ts: theme.extend でトークン参照（colors, borderRadius, minWidth, minHeight, maxWidth）。
- src/app/layout.tsx: next/font で Outfit（本文）・Space Grotesk（ロゴ）を読み込み、html に variable 付与。metadata.icons で icon.svg 指定。
- src/app/components/layout/Header.tsx: 背景 --color-bg-secondary、テキスト/ナビをトークン化。ロゴ「EXER」に Space Grotesk・アクセント色・グロー。
- src/app/components/layout/Footer.tsx: 「SYSTEM ONLINE」「EXER FRAMEWORK」表記、背景・テキストをトークン化。
- src/app/icon.svg: 新規。32×32、角丸矩形、緑 #10b981 の横線3本と右端の円（DS-006）。
- src/app/page.tsx: キャッチコピー「Exercise the Mind, Master the Skill.」、メイン CTA「GET STARTED」で /py-value へ（CP-000）。
- src/app/[workbookId]/QuestionListClient.tsx: カードに --tile-min-width, --radius-lg, トークン色。ソート・タグ・バッジをトークン化。
- src/app/[workbookId]/complete/page.tsx: ボタン・テキストをトークン化。
- src/core/plugins/python-analysis/index.tsx: ワークスペース・採点結果ブロック・ボタンをトークン化（CP-009）。
- src/app/admin/AdminDashboardClient.tsx: ナビ・新規作成ボタン・表・フォームをトークン化。
