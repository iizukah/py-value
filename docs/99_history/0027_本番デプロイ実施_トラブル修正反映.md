## 概要

本番トラブル修正（0026）を反映したうえで、Cloud Build および Firebase により本番デプロイを実施した。

## 実施内容

1. **Cloud Build**: `.env` から ADMIN_KEY と GOOGLE_CLOUD_PROJECT を読み取り、`gcloud builds submit --config=cloudbuild.yaml --substitutions=...` を実行。ビルド成功後、Cloud Run へデプロイ。Service URL: https://exer-next-85028816057.asia-northeast1.run.app
2. **Firebase**: `firebase use exir-27624` のうえで `firebase deploy --only "hosting,firestore"` を実行。Firestore ルール・インデックス（histories 複合インデックス含む）と Hosting をデプロイ済み。

## デプロイ先 URL
