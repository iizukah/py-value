# 0045 インポート用問題データ30問（.drafts 作成）

## 日付
2025-02-09

## 概要
仕様書（DATA-01, DATA-02）に基づき、インポート用問題データを 30 問分、docs/.drafts に JSON ドラフトとして作成した。

## 変更内容
- **新規**: `docs/.drafts/import_questions_30_dataset.json`
  - 問題 30 問（初級 10・中級 10・上級 10）
  - ライブラリ別: NumPy / Pandas / SciPy / sklearn / Matplotlib 各 6 問
  - データセット: `load_iris()`, `load_wine()`, `load_diabetes()` を初期コードでロードし、それに紐づく計算問題
  - 採点: `value_match`、`expected_value` と `tolerance` で数値一致
  - 問題文: Markdown、数式は LaTeX（$...$）
  - 自明な解答禁止のため、すべて計算結果を `ans` に代入する形式
  - 難易度が高いほど統計（t 検定、相関、共分散、交差検証等）・多ステップの計算を要求
